🍽 PETISCOS DE HTML5 🍽

As entradas são os ingredientes principais para qualquer receita de programação. HTML é o esqueleto da receita, fornecendo a estrutura básica.

  1. Petisco: <html>

  2. Petisco: <body>

  3. Petisco: <h1>

🍛 ESSÊNCIA DE ESTILOS 🍛

Os pratos principais são a apresentação da receita. O CSS é o tempero e a aparência que tornam a receita visualmente apetitoso!

  1. Prato: <div>

  2. Prato: <style>

  3. Prato: <span>

🍰 SOBREMESAS INTERATIVAS 🍰

As sobremesas são o toque final da receita. O JavaScript adiciona interatividade ao prato, tornando-o dinâmico e mais "gostoso"!

  1. Sobremesa: <alert()>

  2. Sobremesa: <document.getElementById()>

  3. Sobremesa: <addEventListener()>
